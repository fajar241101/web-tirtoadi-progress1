Course Overview

Pada minggu pertama ini kita akan belajar tentang dasar-dasar tentang menggunakan javascript. Pada minggu kedua kita mulai masuk ke dalam Web Framework Express. Pada minggu ketiga kita akan mulai belajar akses basis data. Pada minggu keempat kita akan fokus Node js lanjutan sebagai backend.


Pada bulan kedua ini kita mendapatkan sebuah project kecil lingkup backend nodejs, kita diberikan kesempatan untuk mengeluarkan ide untuk membuat aplikasi sesuai tema yang diberikan.


Pada bulan ketiga ini kita akan melakukan proyek yang lebih besar dengan mengkolaborasi dengan divisi team lain.
